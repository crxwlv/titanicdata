Women and Children first?

The Titanic sank without enough life boats. There is a belief that the captain said "women and children first." Was it true?

We created a Python program (using the following steps) to answer the question by determining the % of each group (men, women, children) died in the tragedy. Here are the results of my DS project.

  1. Read titanic.csv into a file memory variable
  2. Read the column title row into a string variable
  3. Read the remaining data rows in a list of strings (rows)
  4. Close the file to save computer resources.
  5. Sequential search of each row in the data list
    • Set total and dead counter variables to 0
    • Start the loop
    • Split each row string into list for easy access
    • if the row/person is male, add 1 to total variable
    • if the row/person died, add 1 to dead variable
    • after loop ends, return dead / total * 100
